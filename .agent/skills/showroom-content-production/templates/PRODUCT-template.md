# 商品素材模板 (PRODUCT)

## YAML 元数据

```yaml
type: PRODUCT
title: "[品牌] [产品名] - [核心卖点/系列名]"
productId: "关联商品UUID"
status: PUBLISHED
tags: ["材质", "风格", "适用空间", "颜色", "品牌"]
images:
  - "封面图URL (800x600)"
  - "细节图1 URL"
  - "细节图2 URL"
  - "场景图 URL"
```

```markdown
## 产品亮点

- **[卖点2]**：一句话描述
- **[卖点3]**：一句话描述

```

## 示例

```markdown
## 产品亮点

- **物理遮光 99%**：高精密编织工艺，无需涂层，健康环保
- **垂感极佳**：280g/㎡ 重磅面料，自然下垂，线条优美
- **四季适用**：冬暖夏凉，隔热保温效果显著

## 产品参数

| 参数 | 规格 |
|------|------|
| 材质 | 高精密涤纶 |
| 克重 | 280g/㎡ |
| 门幅 | 2.8m |
| 遮光率 | 99% |
| 工艺 | 高精密编织 |
| 洗涤方式 | 可机洗 (30℃以下) |

## 适用场景

特别适合对遮光要求高的卧室空间。奶咖色调温柔百搭，无论是现代简约还是北欧风格都能完美融入。面料厚实但不沉重，触感柔软细腻。

## 搭配建议

推荐搭配白色或米色纱帘，营造层次感。与木色家具、大地色系软装搭配效果最佳。

## 价格区间

- 标准款：¥120/米
- 电动轨道款：¥180/米起
```
