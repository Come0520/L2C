# 罗莱L2C销售管理系统 - 项目愿景与目标

## 🎯 项目终极目标

### 核心愿景
打造家居软装行业首个**全数字化L2C（Lead to Cash）销售服务平台**，实现从销售线索到现金回款的完整业务链条数字化管理，成为行业数字化转型的标杆项目。

### 战略定位
- **行业地位**：家居软装领域首个完整L2C数字化解决方案
- **技术领先**：采用模块化单体架构，支持渐进式微服务演进
  - **初期（MVP阶段）**：采用模块化单体架构，快速开发和部署
  - **中期（扩展阶段）**：根据业务负载和团队规模，逐步拆分核心服务
  - **成熟期（优化阶段）**：形成完整的微服务架构，支持独立扩展和部署
- **业务创新**：三单体系（销售单→测量单→安装单）状态机驱动
- **生态整合**：整合商品库、服务商、积分激励等完整生态

## 🎨 项目要解决的核心问题

### 1. 销售流程黑盒化问题
**现状痛点**：传统家居销售流程缺乏透明度，客户和内部团队无法实时了解订单状态
**解决方案**：27状态完整流转控制，实时状态同步，多端状态可视化

### 2. 多品类服务协调难题
**现状痛点**：窗帘、墙布、墙咔等多品类服务需要不同的测量师和安装师，协调困难
**解决方案**：技能匹配算法，智能派单系统，服务商绩效管理

### 3. 销售团队激励不足
**现状痛点**：缺乏有效的销售激励机制，销售团队积极性不高
**解决方案**：多节点积分体系，实时积分发放，积分商城兑换

### 4. 数据孤岛问题
**现状痛点**：线索、订单、服务、财务等数据分散，无法进行有效分析
**解决方案**：统一数据平台，L2C销售漏斗分析，多维度数据洞察

## 👥 目标用户群体

### 核心用户
1. **销售团队**
   - 驻店销售：负责门店日常运维和销售，日常门店接单。权限和对应城市或门店绑定
   - 远程销售：协助进群、报价、对接拍单员回传测量数据报价，微信群内和客户或者设计师、导购等确认方案。负责推单处理，上传圣都Home系统采购申请截图。权限可以分配给他的不同城市或门店，获得对应城市的权限
   - 销售负责人：属于该地区的领导，对于订单价格申请有部分权限（折扣权限有限），拥有城市授权分配权限（最高层级分配权限，可跨部门分配）。可以观看查阅系统，不可以更改。权限可以分配给他的不同城市或门店，获得对应城市的权限

2. **交付团队**
   - 订单客服：审核并把订单转录到帘客云系统，并跟踪订单生产进度，及时更新订单生产情况。权限可以分配给他的不同城市或门店，获得对应城市的权限

2. **服务提供商**
   - 测量师：专业测量，提供精准测量服务
   - 安装师：专业安装，确保安装质量
   - 派单员：任务调度，协调服务资源

3. **管理层**
   - 领导：销售管理，团队绩效监控
   - 管理员：系统管理，业务规则配置

### 延伸用户
1. **客户**：通过小程序查看订单状态，确认方案
2. **财务**：对账管理，回款确认
3. **罗莱好伙伴**：协同销售的外部合作伙伴
   - **设计师**：圣都设计师，提供专业设计服务并获取积分激励
   - **导购**：圣都导购人员，协助销售并获取积分激励

## 🌟 主要特性亮点

### 1. 三单体系状态机
- **销售单**：27状态完整流转，支持多品类商品配置
- **测量单**：智能派单，现场数据采集，进度款管理
- **安装单**：安装任务调度，质量验收，尾款收取

### 2. 智能积分激励
- **多节点发放**：线索创建、跟进、转化、成交等节点
- **动态系数**：产品品类、时间段、门店等多维度调节
- **实时到账**：积分实时计算，即时到账提醒

### 3. 技能匹配算法
- **服务商认证**：技能标签，服务质量评分
- **智能派单**：基于技能、地理位置、工作负荷的智能匹配
- **绩效管理**：准时率、质量评分、客诉率统计

### 4. 数据驱动决策
- **L2C漏斗分析**：线索转化率、销售周期、回款分析
- **实时数据看板**：关键指标实时监控，异常预警
- **多维度报表**：销售、服务、财务等多维度分析

### 5. 罗莱好伙伴生态
- **外部合作管理**：设计师、导购等外部伙伴信息管理
- **积分激励体系**：基于订单金额的积分分配和发放机制
- **专属移动端**：好伙伴专属小程序界面，支持积分查询和礼品兑换
- **订单协同**：7天延迟确认机制，确保订单信息准确无误

## 🚀 项目里程碑

### 阶段一：基础平台（MVP）
**时间**：8周
**目标**：完成核心L2C流程，支持线索到回款的完整业务流程
**关键功能**：
- 线索管理与分配
- 销售单27状态流转（详细测量安装环节）
- 积分发放与计算体系
- 积分商城与兑换系统（优先开发）
- 基础报表分析
- 外部测量安装环节采用简易流程（人工填写/表格上传）

### 阶段二：服务生态（扩展）
**时间**：6周
**目标**：完善服务商管理，优化服务流程
**关键功能**：
- 完整的测量安装单管理系统
- 智能派单、技能匹配、服务商绩效管理
- 积分商城功能优化与扩展

### 阶段三：数据智能（优化）
**目标**：数据分析和智能化功能
**关键功能**：
- L2C漏斗分析
- 智能预警
- 数据洞察

### 阶段四：生态整合（成熟）
**目标**：与外部系统深度整合
**关键功能**：VSS系统对接、第三方服务集成、开放API
