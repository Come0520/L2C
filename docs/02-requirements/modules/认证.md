# 认证模块 (Auth)

## 模块定位
负责系统的基础安全能力支撑，包含用户身份验证及相关凭证的生命周期管理。

## 核心功能清单

### 1. 凭证登录 (Login)
- 提供基于邮箱/手机号及密码的登录方式。
- 采用 Glassmorphism 现代视觉交互，并具备防抖、密码可见性切换等细节体验。
- 完善的异常反馈：基于 Zod 严格校验，涵盖凭证错误、网络异常等提示界面展示。

### 2. 密码重置 (Password Reset)
- **重置请求 (`requestPasswordReset`)**：基于收件邮箱生成具备时效性 (15 分钟) 的唯一 token (`uuidv4()`) 并通过底层发送重置凭证邮件给用户。
- **重置执行 (`resetPassword`)**：校验 token 的有效性，包括防复用和过期等检测；校验通过后利用 bcryptjs 进行密码哈希处理，并基于原生事务更新用户主表并标记 token。

## 安全与隔离
- JWT 登录集成 NextAuth.js，并在组件内维护严格的校验逻辑。
- RBAC（角色权限控制，详见 RBAC Guide 扩展说明）。
