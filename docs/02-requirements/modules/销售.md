# 销售模块 (Sales)

## 模块定位
提供面向销售团队全职级的目标管理、业绩追踪与数据洞察支持。模块涵盖目标设定、过程跟进、年/周目标管理以及多维度销售排行榜和完成率分析。

## 核心功能清单

### 1. 销售仪表盘 (Dashboard)
- 汇总当前财月/财年的销售综合数据（金额、完成率等）。
- 区分不同角色的视图（如 admin/manager 查看全局，销售代表查看自身）。

### 2. 销售目标管理 (Sales Targets)
- **月度目标 (`targets.ts`)**：支持查询、更新、目标调整（增减）及目标确认操作。
- **年度目标 (`annual-targets.ts`)**：支持设置年度目标并可按月一键均分拆解至 12 个月模块。
- **周目标 (`weekly-targets.ts`)**：对销售动作进行短周期管理，支持指定周期范围的目标分配及完成进度查询。
- （含个人专属视图获取：`getMySalesTarget`）。

### 3. 数据分析与预警 (Analytics)
- **目标完成率趋势 (`getSalesCompletionTrend`)**：透视近 6 个月历史目标与实际完成金额趋势曲线。
- **销售排名对比 (`getSalesRanking`)**：当月各销售人员业绩目标达成数、金额榜单。
- **月中预警提醒 (`getSalesTargetWarnings`)**：按线性外推法预测月底能否达成目标，识别并预警高、中风险人员。

## 技术依赖
- Drizzle ORM，结合 Zod 和 Action Schema 等安全验证层。
- AuditService 介入对于敏感目标调改的日志审计。
