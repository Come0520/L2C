# 角色与职责 (Roles & Responsibilities)

本文档定义了系统中的核心角色体系及其协作关系。

## [销售人员 (Sales)](./销售人员.md)

- 分为**门店销售**与**驻店销售**。
- 负责线索转化与任务发起（测量/安装）。
- 支持“指定指派”或“委托派单”。

## [派单员 (Dispatcher)](./派单员.md)

- 销售与工人之间的枢纽。
- 负责处理未指定工人的任务池。
- 负责工费谈判与成本管控。

## [工人 (Workers)](./工人.md)

- 包含**测量师**与**安装师**。
- 负责现场执行与数据回传。
