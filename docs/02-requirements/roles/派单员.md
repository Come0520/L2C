# 派单员 (Dispatcher)

## 角色定位

连接“前端销售”与“后端交付”的桥梁，负责资源的统筹调度与成本管控。

## 核心职责

1.  **任务调度**:
    - 监控“待派单池”中的测量与安装任务。
    - 根据工人位置、技能、空闲时间进行智能或人工派单。
2.  **价格谈判**:
    - 作为销售和工人之间的议价缓冲。
    - 核算工费成本，确保利润空间。
    - 与工人沟通特殊工艺的额外费用。
3.  **进度监控**: 跟踪任务执行状态，处理工单异常（如工人爽约、无法联系客户）。

## 交互流程

- **接收需求**: 收到销售提交的未指定工人的需求单。
- **指派工人**: 在系统中选择合适的测量师/安装师，点击“派单”。
- **费用确认**: 在派单前或完工后，确认应付工费。
