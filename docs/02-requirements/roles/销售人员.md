# 销售人员 (Sales)

## 角色概述

销售人员是一个人员类别，包括销售、店长等与客户直接接触、负责业务拓展和订单管理的角色。

## 角色类型

1.  **门店销售 (Store Sales)**: 驻守在展厅或门店，接待自然进店或邀约到店的客户。
2.  **驻店销售 (Stationed Sales)**: 派驻在合作渠道（如装修公司、样板间）的销售人员。
3.  **店长 (Store Manager)**: 门店负责人，负责门店整体运营、团队管理和业务审批。

## 角色职责

### 门店销售 / 驻店销售

- **线索转化**: 负责从线索 (Lead) 到成交的全过程跟进。
- **需求提交**: 发起测量 (Measure) 和安装 (Install) 需求。
- **报价与签约**: 根据测量结果生成报价单，并引导客户签约支付。
- **收款管理**: 创建收款单，上传收款凭证，跟进收款进度。

### 店长

- **团队管理**: 管理销售团队，分配任务，监督工作进度。
- **业务审批**: 审核报价单、收款单、付款单等业务单据。
- **财务管理**: 审核收款凭证和付款凭证，确认到账和付款。
- **数据监控**: 查看门店业绩数据，分析业务趋势，制定改进策略。
- **权限控制**: 管理系统权限，配置业务规则和系统设置。

## 协作流程

- **发起任务**:
  - 在提交测量/安装需求时，若已有长期合作或客户指定的师傅，可直接指派。
  - 若无指定人选，则提交至“待派单池”，由派单员介入协调。
- **报价协同**: 接收测量师上传的数据，生成最终报价。
