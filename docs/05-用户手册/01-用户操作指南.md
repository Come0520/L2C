## 系统概述

### 系统简介
罗莱L2C销售管理系统是专为家居软装行业打造的全数字化销售服务平台，通过三单体系（销售单→测量单→安装单）实现从线索到现金的完整业务流程数字化管理。

### 主要功能模块
- **线索管理**：线索录入、分配、跟踪、转化
- **销售单管理**：销售单创建、状态流转、合同管理
- **测量单管理**：测量任务派单、进度跟踪、数据采集
- **安装单管理**：安装任务调度、质量验收、完工确认
- **积分系统**：积分发放、商城兑换、排行榜展示
- **服务商管理**：技能认证、绩效管理、任务分配

### 系统特色
- **全流程数字化**：从线索获取到销售单完成的全程跟踪
- **智能积分体系**：激励员工积极性，提升工作效率
- **移动端支持**：支持手机端操作，随时随地办公
- **数据可视化**：丰富的报表和图表，直观展示业务数据

## 线索管理操作

### 线索录入
1. 点击"线索管理" → "新增线索"
2. 填写客户基本信息：
   - **客户姓名**（必填）：客户真实姓名
   - **联系电话**（必填）：主要联系方式
   - **客户地址**（必填）：详细地址信息
   - **意向产品**：窗帘、墙纸、墙板等
   - **预算范围**：客户预期投入金额
   - **线索来源**：电话咨询、网络推广、门店到访等
   - **客户需求**：具体需求描述
3. 选择负责销售人员
4. 设置跟踪提醒时间
5. 点击"保存"完成录入

### 批量导入线索
1. 点击"线索管理" → "批量导入"
2. 下载Excel模板
3. 按模板格式填写线索信息：
   - 确保必填字段完整
   - 电话号码格式正确
   - 地址信息详细准确
4. 上传Excel文件
5. 系统自动验证并导入
6. 查看导入结果报告

### 线索跟踪
1. 在线索列表中找到目标线索
2. 点击"跟踪记录"
3. 添加跟踪内容：
   - **跟踪时间**：实际沟通时间
   - **沟通方式**：电话/微信/上门/邮件
   - **沟通内容**：详细记录沟通过程
   - **客户反馈**：客户的回应和态度
   - **客户状态**：意向程度评估
   - **下次跟踪时间**：计划下次联系时间
4. 上传相关附件（如聊天截图）
5. 保存跟踪记录

### 线索分配
1. 选择需要分配的线索
2. 点击"批量分配"
3. 选择目标销售人员
4. 填写分配说明
5. 确认分配操作
6. 系统自动通知相关人员

### 线索转化
1. 确认客户有明确购买意向
2. 点击"转为销售单"
3. 系统自动创建销售单
4. 线索状态更新为"已转化"
5. 销售人员获得转化积分

## 销售单管理操作

### 创建销售单
1. 点击"销售管理" → "新增销售单"
2. 填写基本信息：
   - **客户信息**：可从线索导入或手动录入
   - **销售人员**：负责销售人员
   - **门店信息**：所属门店
   - **预计交付时间**：客户期望时间
3. 添加商品明细：
   - 选择商品品类（窗帘/墙纸/墙板）
   - 输入商品规格和数量
   - 设置单价和折扣
   - 系统自动计算总金额
4. 设置付款方式和条件
5. 保存销售单

### 销售单状态流转
销售单包含27个状态，按业务流程依次流转：

**线索阶段状态（5个）：**
1. **待分配** - 新线索等待分配
2. **待跟踪** - 已分配待跟踪
3. **跟踪中** - 正在跟踪联系
4. **草签** - 客户初步确认意向
5. **已关闭** - 线索关闭

**销售单阶段状态（15个）：**
6. **待测量** - 等待安排测量师上门测量
7. **测量中-待分配** - 测量需求单已生成，等待派单员分配
8. **测量中-分配中** - 派单员正在分配测量师
9. **测量中-待上门** - 测量师已接单，等待上门测量
10. **测量中-待确认** - 测量完成，等待销售确认测量单
11. **方案待确认** - 测量单已确认，等待客户确认方案
12. **待推单** - 方案已确认，等待推单处理（远程销售上传圣都Home系统采购申请截图）
13. **待下单** - 推单确认完成，等待下单（采购申请已确认，进入生产环节）
14. **生产中** - 销售单已下单，正在生产
15. **备货完成** - 所有生产单号备货完成
16. **安装中-待分配** - 安排安装，等待派单员分配安装师
17. **安装中-分配中** - 派单员正在分配安装师
18. **安装中-待上门** - 安装师已接单，等待上门安装
19. **安装中-待确认** - 安装完成，等待销售确认
20. **待对账** - 安装确认完成，等待财务对账

**财务阶段状态（2个）：**
21. **待回款** - 财务已填写发票单号，等待客户回款
22. **已完成** - 回款确认完成，销售单结束

**异常状态（5个）：**
23. **已取消** - 销售单被取消
24. **暂停** - 销售单暂停处理
25. **异常** - 销售单出现异常情况
26. **暂停** - 销售单暂停处理  
27. **异常** - 销售单出现异常情况

### 状态操作说明
- **推送测量**：销售单确认后，推送给测量师
- **生成报价**：测量完成后，系统生成正式报价
- **客户确认**：客户确认报价，进入下单状态
- **安排生产**：销售单进入生产排期
- **发货通知**：产品发货，通知客户
- **完成安装**：安装师完成现场安装
- **客户验收**：客户确认安装质量
- **完成回款**：财务确认款项到账

### 销售单编辑
1. 在草稿或草签状态下可以编辑
2. 修改商品信息和数量
3. 调整价格和折扣
4. 更新客户需求
5. 保存修改内容

## 测量单管理操作

### 接收测量任务
1. 测量师登录系统
2. 查看"我的任务"列表
3. 查看任务详情：
   - 客户信息和联系方式
   - 测量地址和时间要求
   - 特殊需求说明
4. 点击"接受任务"确认接单
5. 联系客户预约具体测量时间

### 现场测量操作
1. 到达客户现场，确认身份
2. 点击"开始测量"
3. 录入测量数据：
   - **房间信息**：房间名称、用途
   - **尺寸数据**：长、宽、高精确测量
   - **窗户规格**：窗户尺寸、类型、朝向
   - **墙面情况**：墙面材质、平整度
   - **特殊要求**：客户特殊需求记录
4. 拍摄现场照片（5-15张）：
   - 整体房间照片
   - 窗户细节照片
   - 墙面情况照片
   - 特殊位置照片
5. 客户确认测量数据
6. 客户签字确认
7. 点击"完成测量"

### 测量数据提交
1. 检查测量数据完整性
2. 确认所有照片已上传
3. 填写测量备注和建议
4. 提交测量报告
5. 系统自动通知销售人员
6. 生成测量单PDF文档

### 测量异常处理
1. **客户不在家**：联系客户重新预约
2. **现场条件不符**：记录实际情况，联系销售
3. **测量数据异常**：重新测量确认
4. **客户要求变更**：记录变更内容，通知销售

1. 安装师登录系统

2. 准备安装工具和材料

1. 完成所有安装工作

   - 检查安装质量
   - 确认功能正常

4. 拍摄完工照片
5. 客户签字确认
6. 提交完工报告

### 安装售后服务
1. 向客户介绍使用方法

4. 记录客户反馈

- **线索转化**：+50积分

## 常见问题解答

### Q1：忘记密码怎么办？
**A：** 
1. 点击登录页面"忘记密码"链接
2. 输入用户名或手机号
3. 获取验证码重置密码
4. 或联系系统管理员重置密码

### Q2：线索导入失败怎么处理？
**A：**
1. 检查Excel格式是否正确
2. 确认必填字段是否完整
3. 验证电话号码格式
4. 检查数据是否重复
5. 联系技术支持获取帮助

### Q3：积分没有及时到账怎么办？
**A：**
1. 积分通常在操作完成后5分钟内到账
2. 检查操作是否符合积分规则
3. 查看积分明细记录
4. 如有延迟请联系客服处理

### Q4：测量数据录入错误如何修改？
**A：**
1. 在测量单提交前可以直接修改
2. 提交后需要联系管理员处理
3. 重新测量需要客户确认
4. 修改记录会保留在系统中

### Q5：系统运行缓慢怎么办？
**A：**
1. 清除浏览器缓存和Cookie
2. 检查网络连接状态
3. 关闭不必要的浏览器标签
4. 重启浏览器或设备
5. 联系技术支持检查服务器状态

### Q6：如何提高线索转化率？
**A：**
1. 及时跟踪线索，保持客户热度
2. 详细记录客户需求和偏好
3. 提供专业的产品建议
4. 建立良好的客户关系
5. 定期回访和关怀

### Q7：手机端如何使用系统？
**A：**
1. 使用手机浏览器访问系统网址
2. 系统支持响应式设计，自动适配手机屏幕
3. 核心功能在手机端都可以正常使用
4. 建议使用微信小程序版本（如已开发）

\n+## 功能使用指南
\n+### 开发模式访问
在开发环境下，系统已配置为跳过登录界面，直接访问 http://localhost:3000 即可进入线索管理系统。
\n+### 线索管理
\n+#### 查看线索列表
1. 访问主页面，默认显示线索列表
2. 支持按状态、优先级、来源等条件筛选
3. 支持按姓名、电话等关键词搜索
4. 点击列表项可查看详细信息
\n+#### 创建新线索
1. 点击"新建线索"按钮
2. 填写必要信息：
   - 姓名（必填）
   - 电话（必填）
   - 邮箱（可选）
   - 来源（必选）
   - 优先级（必选）
   - 备注（可选）
3. 点击"保存"完成创建
\n+#### 编辑线索信息
1. 在线索列表中点击"编辑"按钮
2. 修改相关信息
3. 点击"保存"确认修改
\n+#### 线索状态管理
线索状态包括：
- **新线索**: 刚创建的线索
- **已联系**: 已经进行过沟通
- **有意向**: 客户表现出购买意向
- **已成交**: 成功转化为客户
- **已失效**: 无效或失去联系的线索
\n+状态变更：
1. 在线索详情页面选择新状态
2. 添加状态变更备注
3. 确认状态更新
\n+#### 快速跟进
1. 在线索列表或详情页点击"快速跟进"
2. 选择预设的跟进模板或自定义内容
3. 设置下次跟进时间
4. 保存跟进记录
\n+### 批量操作
\n+#### 批量分配
1. 在线索列表中勾选多个线索
2. 点击"批量分配"按钮
3. 选择分配的销售人员
4. 确认分配操作
\n+#### 批量状态更新
1. 勾选需要更新的线索
2. 点击"批量操作" > "更新状态"
3. 选择新状态
4. 添加批量操作备注
5. 确认更新
