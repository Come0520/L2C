## 移动端概述

### 产品简介
罗莱L2C销售管理系统移动端是专为销售人员、测量师、安装师等一线工作人员设计的移动应用，支持随时随地处理业务，提高工作效率。

### 主要功能
- **线索管理**：外出时快速录入和跟踪客户线索
- **销售管理**：移动端处理销售单和客户沟通
- **任务管理**：接收和处理测量、安装任务
- **实时通讯**：即时消息和任务通知
- **数据同步**：与PC端数据实时同步
- **离线支持**：网络不佳时支持离线操作

### 支持平台
- **微信小程序**：无需安装，即用即走
- **iOS App**：支持iOS 12.0及以上版本
- **Android App**：支持Android 8.0及以上版本
- **H5网页版**：支持主流移动浏览器

## 安装与登录

### 微信小程序版本

#### 首次使用
1. 打开微信，搜索"罗莱L2C"小程序
2. 点击进入小程序
3. 授权微信登录
4. 输入员工工号和密码
5. 绑定微信账号

#### 快速进入
1. 微信下拉，在"最近使用"中找到小程序
2. 或在微信"发现" → "小程序"中查找
3. 可添加到桌面快捷方式

### App版本安装

#### iOS安装
1. 打开App Store
2. 搜索"罗莱L2C"
3. 点击"获取"下载安装
4. 安装完成后打开应用

#### Android安装
1. 扫描二维码下载APK文件
2. 允许安装未知来源应用
3. 点击安装包进行安装
4. 安装完成后打开应用

### 登录流程
1. 输入登录信息：
   - **工号**：员工唯一标识
   - **密码**：登录密码
   - **验证码**：图形验证码（如需要）
2. 选择"记住密码"（可选）
3. 点击"登录"
4. 首次登录需要：
   - 完善个人信息
   - 设置安全问题
   - 开启消息推送

## 线索管理

### 线索录入

#### 快速录入
1. 点击首页"新增线索"
2. 填写基本信息：
   - **客户姓名**：必填
   - **联系电话**：必填
   - **客户地址**：可选择地图定位
   - **需求描述**：产品类型和需求
3. 选择线索来源：
   - 电话咨询
   - 门店到访
   - 网络推广
   - 朋友推荐
4. 点击"保存"完成录入

#### 详细录入
1. 点击"详细录入"
2. 补充完整信息：
   - **客户信息**：年龄、职业、收入等
   - **房屋信息**：面积、装修状态、风格
   - **需求信息**：预算、时间、具体要求
   - **竞争信息**：是否对比其他品牌
3. 上传相关图片：
   - 房屋照片
   - 现场测量图
   - 参考样式图
4. 设置跟踪提醒
5. 保存线索信息

### 线索跟踪

#### 跟踪记录
1. 进入线索详情页
2. 点击"添加跟踪"
3. 选择跟踪方式：
   - 电话沟通
   - 微信聊天
   - 上门拜访
   - 邮件联系
4. 记录沟通内容：
   - 客户反馈
   - 需求变化
   - 竞争情况
   - 下次跟踪计划
5. 上传相关附件
6. 保存跟踪记录

#### 快速操作
1. **一键拨号**：直接拨打客户电话
2. **发送短信**：快速发送产品信息
3. **微信联系**：跳转微信聊天
4. **导航到访**：地图导航到客户地址

### 线索转化

#### 转化为销售单

#### 手动更新
1. 在销售单详情页
2. 点击"更新状态"
3. 选择新状态：
   - 待测量
   - 测量中
   - 待报价
   - 已报价
   - 待签约
   - 已签约
   - 生产中
   - 待安装
   - 安装中
   - 已完成

### 现场测量

#### 测量准备
1. 确认测量工具：
   - 激光测距仪
   - 卷尺
   - 水平仪
   - 相机
2. 联系客户确认时间
3. 导航到客户地址

   - **房间尺寸**：长、宽、高

   - 注意事项

## 常见问题

### 登录问题

#### 无法登录
**问题**：输入正确账号密码但无法登录
**解决方案**：
1. 检查网络连接是否正常
2. 确认账号密码是否正确
3. 检查是否被管理员禁用
4. 清除应用缓存重新登录
5. 联系管理员重置密码

#### 登录超时
**问题**：登录过程中提示超时
**解决方案**：
1. 检查网络信号强度
2. 切换网络环境（WiFi/4G）
3. 重启应用重新登录
4. 检查服务器状态

### 数据同步问题

#### 数据不同步
**问题**：移动端数据与PC端不一致
**解决方案**：
1. 手动触发数据同步
2. 检查网络连接状态
3. 重启应用重新同步
4. 清除缓存重新登录
5. 联系技术支持

#### 同步失败
**问题**：数据同步一直失败
**解决方案**：
1. 检查网络连接稳定性
2. 确认服务器状态正常
3. 检查数据格式是否正确
4. 分批次同步大量数据
5. 联系技术支持处理

### 功能使用问题

2. 检查存储空间是否充足
3. 压缩图片大小后重新上传
4. 检查网络连接状态
5. 重启应用重新尝试

#### 定位不准确
**问题**：地址定位不准确或无法定位
**解决方案**：
1. 检查定位权限是否开启
2. 确认GPS功能是否启用
3. 在空旷地区重新定位
4. 手动输入详细地址
5. 重启设备重新定位
