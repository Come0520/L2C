# notifications (神经反射与消息总线) 模块价值挖掘与卖点报告

> 挖掘日期：2026-02-23
> 目标模块：`src/features/notifications/`

## 🎯 核心定位 (One-Liner)

**L2C 的神经反射弧：把人找事变成“事人合一”。不论你在世界的哪个角落，最隐秘或者最紧急的业务炸弹，系统 1 秒自动飞到你眼前。**

## 💎 核心卖点清单 (Killer Features)

### 1. 全域立体消息触达 (Omnichannel Bombardment)

- **技术支撑**: 基于 Redis 和 Edge 边缘环境分发的极速 WebSocket 长连接以及微信/短信集成。
- **业务价值**: 老总去三亚度假了，店里发生一笔超大额异常退款怎么办？系统立刻在网页端飘窗、在手机端弹窗、在微信端推送。不需要任何人给老板打电话汇报，坏消息总是跑得比内鬼的嘴巴还要快。
- **话术金句**: “不管是特大喜报还是严重警报：L2C 总能在第一时间找到能拍板的那个人。”

### 2. 精准降噪与“只看干货”收件箱 (Noise-Canceling Inbox)

- **技术支撑**: `markAsRead`, 消息类型 `Category` 降级屏蔽与分类 (Info/Warning/Error/Success)。
- **管理价值**: 老总不想每天被几百条“张三发了份报价”的垃圾短信烦死？可以在设置里屏蔽常规通知，只放行“逾期三天未收款”、“客诉红线爆发”、“触发退款阻断”等爆炸性事件。
- **话术金句**: “过滤所有业务闲言碎语，只听见最致命的金钱落地声。”

### 3. 基于上下文的极速决策直达 (Contextual Action Links)

- **技术支撑**: 每条通知不仅带内容，还附带了直达底层业务界面的深层路由 (Deep Linking) 与参数。
- **效率价值**: 当主管收到“王五给客户报了一个惊人折扣请求审批”，不需要主管再去登录系统输入账号慢慢去翻。他点开消息卡片，整个打款截图和审批按钮已经在他指尖。秒赞，秒拒。
- **话术金句**: “每一个弹窗都是一张签批单，让流程运转的速度快过员工呼吸的频率。”

## 🎬 场景化故事 (User Stories)

### 场景一：把一笔死账从火线上捞回来

- **痛点**: 业务员忘了催尾款，财务不敢催客户，店长每天忙成陀螺根本不知道哪笔单子该进钱了。钱就这么拖没了。
- **解决方案**: L2C 逾期机器人。尾款倒计时只要到期，系统的催款警报同时在财务电脑、店长手机、甚至当事业务员的工作群里狂闪。如果再拖延 3 天，警报连升两级直接发进老总收件箱。老板只需点开通知：“尾款 6 万，责任员工张三”，一个质问电话拨过去。
- **结果**: 全公司的每一滴死水都被激活了，账龄坏账率断崖式下滑。
