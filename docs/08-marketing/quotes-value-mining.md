# quotes 模块价值挖掘与卖点报告

> 挖掘日期：2026-02-23
> 目标模块：`src/features/quotes/`

## 🎯 核心定位 (One-Liner)

**L2C 智能算价与成单加速器：告别糊涂账，将经验固化为算法的报价印钞机。**

## 💎 核心卖点清单 (Killer Features)

### 1. 策略化精准算价大脑 (Strategy-Based Calculation Engine)

- **技术支撑**: `StrategyFactory` 动态匹配算价策略（如帘头、罗马杆、布艺不同算法），`recalculateQuote()` 实现一键全局重算。
- **老板价值**: 彻底告别人工手算带来的“少算亏本、多算跑单”。复杂的褶皱比例、面料门幅、工艺损耗全由系统精准计算，毛利牢牢抓在手里。
- **执行价值**: 销售不再需要背熟复杂的算价公式，输入宽高和款式，系统 1 秒出精确报价单。
- **话术金句**: “把资深算价师的经验装进系统，让新人第一天就能开出零差错的复杂大单。”

### 2. 多版本对比与时光机 (Multi-Version Quote Management)

- **技术支撑**: `createNextVersionAction()` 引用原单创建新版本，同时利用 `version` 乐观锁防并发冲突。
- **老板价值**: 记录客户的每一次预算博弈。无论客户怎么改需求，都能随时调出历史版本进行对比，防守利润底线。
- **管理价值**: 每次修改都有迹可循，杜绝“改到最后不知道用的哪一版”的管理灾难。
- **话术金句**: “不怕客户改方案，每一次修改都是一次精准的成单演练。”

### 3. 一键克隆的爆款模板库 (Quote Templates)

- **技术支撑**: `saveQuoteAsTemplate()` 沉淀模板，`createQuoteFromTemplate()` 快速复用。
- **管理价值**: 将公司最畅销、利润最高的搭配套餐固化为全员共享模板，标准化输出公司核心产品。
- **执行价值**: 全屋套餐一键套用，只要改改尺寸就能出单，将原来 半小时 的做单时间压缩至 3 分钟。
- **话术金句**: “复刻销冠的超级套餐，让每一次报价都自带成单基因。”

### 4. 严密的利润护城河 (Discount Approval & Lifecycle)

- **技术支撑**: `approveQuote()`, `rejectQuoteDiscount()`, 带有细颗粒度 `PERMISSIONS.QUOTE.APPROVE` 校验的审批流。
- **老板价值**: 锁死最低折扣率。低于底价的订单必须老板或店长审批，坚决杜绝私放折扣损害公司利润。
- **管理价值**: 从草稿、提交、锁网到审批，完整的生老病死状态机管理，流转清晰。

## 🎬 场景化故事 (User Stories)

### 场景一：惊心动魄的三次改稿

- **痛点**: 客户全屋定制，先选了真丝，觉得贵换成棉麻，最后又想看天鹅绒的价格。以前销售要手算三遍，焦头烂额还容易算错。
- **解决方案**: L2C 报价系统支持“多版本创建”，点一下复制，换个材质，1 秒重出总价。三个版本发给客户对比，清清楚楚。
- **结果**: 客户感受到极高的专业度，当场定下第二版方案。
