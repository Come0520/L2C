# showroom (云展厅) 模块价值挖掘与卖点报告

> 挖掘日期：2026-02-23
> 目标模块：`src/features/showroom/`

## 🎯 核心定位 (One-Liner)

**L2C 裂变拓客雷达：不仅仅是能装进口袋的电子图册，更是能捕获高意向客户的隐形销售员。**

## 💎 核心卖点清单 (Killer Features)

### 1. 千人千面的专属分享链 (Dynamic Trackable Shares)

- **技术支撑**: `createShareLink()` 组合展品生成一个独立的、带有过期时间戳及定制价格策略（改价快照）的短链接。
- **业务价值**: 线下给客户报价容易成为同行比价的靶子！有了云展厅，销售给客户发送的是带时效性的阅后即焚链接。
- **执行价值**: A 客户觉得 3 万全包贵了，销售不需要改系统价格打乱原本的价格体系，直接用这套家具创建一个“特惠打折版”独立分享链接发送给 A 客户看，一人一个价。
- **话术金句**: “随时给客户发一套量身定制的方案配价格，比纸质画册高级 100 倍。”

### 2. 精准的访客活动雷达 (Visitor Activity Tracking)

- **技术支撑**: `getShareContent()` 集成严格限流与 Redis 抗压计数，采样回写客户在 C 端的每一次点击与浏览深度数据。
- **老板价值**: 传统发个朋友圈图片，根本不知道谁看了。而云展厅发群里，后台能立刻收到反馈：谁点开了链接，谁看了 3 遍这款三人沙发，谁停留了 5 分钟。
- **执行价值**: 销售收到系统提示：“微信昵称为李女士的客户连续翻看了 5 遍《奶油风套餐》”，这时候打个回访电话，成单率直线上升！
- **话术金句**: “发出去的不是商品链接，是一个帮你 24 小时探测出高意向客户的隐形雷达。”

### 3. 全链路大盘宣发罗盘 (Share Dashboard Statistics)

- **技术支撑**: `getShareDashboardStats()` 汇聚整个租户下的分享记录、访问总量和漏斗转化率。
- **管理价值**: 公司上了一批新品，让大家发朋友圈。以前不知道员工到底发没发、效果好不好。现在后台罗盘一转，哪个产品被分享最多最带量、哪个员工分享拉新的客户数量最高，一切清清楚楚。
- **营销价值**: 依据点击热度排行榜指导采购备货，什么款火进什么材料，不压库不高评。
- **话术金句**: “把门店的商品推广拉入数字化时代，每一次点击浏览都在为采购选品投赞成票。”

## 🎬 场景化故事 (User Stories)

### 场景一：洞悉客户真实意向

- **痛点**: 小张接待了一个大哥，聊了一小时没下单走了。加了微信后小张发了一堆高清无码材料图，大哥连回都不回，小张觉得这单死定了。
- **解决方案**: L2C 指导小张利用云展厅功能。小张将大哥看中的窗帘套餐打包成一条 “李哥专享看样链接”，并设置只活 2 天，微信发了过去。晚上 9 点，系统雷达推送通知：“李先生刚点开了您的分享，并在‘进口绒布主卧套餐’停留了 3 次共 15 分钟。” 小张心里有底了。
- **结果**: 第二天小张以此入手进行沟通，提出给卧室做个升级服务，成功破冰签下单子。大哥惊讶：“哎你怎么知道我昨晚一直看那个卧室窗帘的？”
