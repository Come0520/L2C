## 快速切换主题脚本

如果您想快速切换主题验证，可以在浏览器 Console 中运行：

```javascript
// 粘贴到 Chrome DevTools Console
const themes = ['theme-warm', 'theme-glass', 'theme-linear'];
let currentIndex = 0;

// 按 Ctrl+T 切换主题
document.addEventListener('keydown', (e) => {
  if (e.ctrlKey && e.key === 't') {
    currentIndex = (currentIndex + 1) % themes.length;
    document.body.className = themes[currentIndex];
    console.log(`✅ 切换到: ${themes[currentIndex]}`);
  }
});

console.log('👍 主题快捷键已启用！按 Ctrl+T 切换主题');
```
