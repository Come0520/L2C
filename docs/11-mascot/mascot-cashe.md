# 吉祥物设定：Cashe (小C)

## 1. 基本信息

- **英文名**: Cashe
- **中文名**: 凯丝 / 小C
- **物种**: 苏格兰折耳猫
- **体型/外貌**: 体型极其圆润，**肚子非常大、圆滚滚的**，看起来软糯可爱，极具亲和力。像人类一样**两脚直立行走**（拟人化站姿），方便用前爪拿取测量工具和嗑瓜子。
- **职业**: 窗帘行业全能人员（业务/测量/设计）
- **核心使命**: 完美执行从“线索 (Lead)”到“现金 (Cash)”的全流程 (L2C)。

## 2. 核心视觉特征与“彩蛋”

小C的形象设计深度融合了 L2C 系统的核心理念：

- **右眼 (虹膜与瞳孔 C)**: 保留正常的白色眼白。虹膜为温暖的橘黄色，正中心的黑色瞳孔呈现漫画风的字母 `C` 形状，代表 **C**ash（现金）。
- **左眼 (虹膜与瞳孔 L)**: 保留正常的白色眼白。虹膜为明亮的绿色，正中心的黑色瞳孔呈现漫画风的字母 `L` 形状，代表 **L**ead（线索）。
- **肚皮暗纹 (白色绒毛2)**: 她的整个圆肚子本身也是蓝色的，但在正中央，长着一片超级明显的、形状像数字 `2` 的白色毛茸茸绒毛斑块，巧妙连接左眼的 L 和右眼的 C。
- **专属配饰**:
  - **皮尺项圈**: 不是普通的猫咪项圈，而是用窗帘行业必不可少的测量皮尺做成，方便她随时随地上门测量尺寸。
  - **水桶包**: 斜挎一个有质感的棕色牛皮水桶包（百宝箱），包口微微露出她的工作法宝——**图纸**、**账本**，可能还有一小袋**瓜子**。

## 3. 性格设定 (Personality)

- **外表慵懒，内心精明**: 圆滚滚的体型让她看起来慢吞吞、贪睡，但听到“新线索”或金钱划账的声音，双眼（L和C）会瞬间放大，立刻进入高效专业的战斗状态。
- **“细节控”强迫症**: 对数字和尺寸极端敏感。看到歪斜的窗框或测量数据有一丝误差，就会下意识拉拉脖子上的皮尺重新复核。
- **无敌亲和力的金牌销售**: 自带蓝猫的温柔属性，加上圆滚滚非常容易让人想rua的大肚子，用软萌可爱化解谈单中的僵局，极易获取挑剔客户的信任。

## 4. 标志性动作与小怪癖 (Quirks & Actions)

- **爱嗑瓜子**: 最喜欢的零食是瓜子。休息时或者思考方案时，喜欢两脚站立靠在窗边，用肉乎乎的前爪灵活地嗑瓜子，偶尔还会把瓜子壳藏在水桶包的最底层。
- **扯皮尺**: 思考方案、遇到难题或计算报价时，会用山竹般的小爪子去拉动脖子上的皮尺，“唰唰”作响是她大脑飞速运转的信号。
- **护包狂魔**: 每次谈成一单，会小心翼翼地把更新后的账本和图纸塞回水桶包，并满意地拍拍包侧。
- **异瞳微表情**:
  - _左眼（绿眼 L）发亮_: 嗅到优质客户或新线索。
  - _右眼（橘眼 C）发亮_: 成功收单，钱款到账。
  - _无语/遇到棘手事情_: 两只眼睛的 L 和 C 会变成半眯着的“无语”形状。
- **布料收集癖**: 痴迷各种优质窗帘布料（丝绒、雪尼尔等），遇到好料子会忍不住把圆脸埋进去狂蹭。

## 5. 交互状态 (UI/交互理念)

在 L2C 系统中，小C可作为智能 Copilot 或吉祥物出现在各类场景中：

- **加载/抓取线索时 (Loading)**: 从一堆毛线球（线索池）里精准扒拉出目标线索，或者一边嗑瓜子一边翻阅资料。
- **报价/计算时 (Calculations)**: 戴上黑框老花镜（或拨弄微型算盘），疯狂翻阅水桶包里的账本。
- **成交/收款时 (Success)**: 眼里闪烁星星，开心地把水桶包高高举起（撒花），挺起圆滚滚的白肚皮。

## 6. 宣传口号 (Catchphrases)

- “喵~ 每一厘米的精准，都是为了最后的 Cash！”
- “你的线索（指左眼），我的现金（指右眼），还有我的瓜子（拍拍水桶包）！”
- “别担心，不管是复杂的飘窗还是挑高的复式，交给小C和我的水桶包准没错！”
