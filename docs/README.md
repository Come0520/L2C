## 📋 文档概述

本文档库包含罗莱L2C销售管理系统的完整需求文档，涵盖从项目规划到用户手册的全生命周期文档。文档按照项目实施阶段进行分类组织，便于不同角色的人员快速找到所需信息。

**项目名称：** 罗莱L2C销售管理系统  
**项目类型：** 家居软装行业数字化销售服务平台  
**技术架构：** 前后端分离 + 模块化单体架构  
**核心业务：** Lead to Cash (L2C) 全流程数字化管理

### 技术架构概览
```
技术架构层次
├── 前端层
│   ├── PC端：Next.js 15+ + Tailwind CSS
│   ├── 移动端：Taro 3.x + React
│   └── 管理端：Next.js 15+ + Ant Design
├── 网关层
│   ├── API网关：Kong/Nginx
│   ├── 负载均衡：Nginx
│   └── 安全防护：WAF
├── 应用层
│   ├── 业务服务：Node.js + Express
│   ├── 认证服务：JWT + Redis
│   └── 文件服务：OSS + CDN
├── 数据层
│   ├── 关系数据库：PostgreSQL 15+
│   ├── 缓存数据库：Redis 6.0
│   └── 文件存储：阿里云OSS
└── 基础设施层
    ├── 容器化：Docker + Docker Compose
    ├── 监控运维：Prometheus + Grafana
    └── 日志分析：ELK Stack
```
