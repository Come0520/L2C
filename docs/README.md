## 🔍 快速查找

- **想了解某个模块的需求？** → `02-requirements/modules/`
- **想看升级计划或 PRD？** → `03-plans/`
- **想查开发规范或配置字典？** → `04-guides/`
- **想查某个模块的审计结果？** → `06-audits/<模块名>/`
- **想评估模块成熟度？** → `07-maturity-reports/`
- **想提取模块卖点？** → `08-marketing/`
- **查看吉祥物 Cashe 相关资产？** → `11-mascot/`
