## 🚀 快速入门

### 您的角色权限

作为**驻店销售（SALES_STORE）**，您可以：
- ✅ 接收并跟踪分配给您的线索
- ✅ 添加跟进记录、约见客户
- ✅ 生成报价单、确认草签
- ✅ 安排测量、确认方案
- ✅ 推单下单、跟踪订单
- ❌ 不能分配线索（这是销售主管/渠道的权限）

### 每日工作流

```
早上上班
  ↓
1. 查看"待跟踪"线索（立即处理）
2. 查看预警提醒（优先级处理）
3. 跟进"跟踪中"线索
4. 处理"草签"客户（安排测量）
  ↓
白天工作
  ↓
5. 到店接待客户
6. 添加跟踪记录
7. 生成报价单
  ↓
下班前
  ↓
8. 更新线索状态
9. 设置明日提醒
```

## 🔄 线索管理全流程

### 阶段 1：接收线索

#### 场景：主管给您分配了一个新线索

**操作步骤**：

1. **收到系统提醒**
   - 您的线索列表会显示一条"待跟踪"状态的线索
   - 线索编号如：`L20251212001`

2. **查看线索详情**
   - 点击线索，进入详情页
   - 查看客户基本信息：
     - 姓名、电话
     - 项目地址
     - 预算范围
     - 来源渠道
     - 客户等级（A/B/C/D）

3. **⏰ 时效要求**
   - ⚠️ **72小时内必须开始跟踪**
   - ⏰ 48小时时系统会提醒您
   - 🚫 超过120小时未跟踪：扣5分 + 线索退回

**操作提示**：
```
💡 立即点击"开始跟踪"按钮
→ 状态变更为"跟踪中"
→ 启动跟踪计时
```

---

### 阶段 2：跟踪客户

#### 2.1 首次联系

**目的**：确认客户需求，建立信任

**操作步骤**：

1. **拨打电话**
   ```
   话术参考：
   "您好，我是XXX公司的XX，您之前咨询过我们的
   产品，想了解一下您现在的装修进度..."
   ```

2. **添加跟踪记录**
   - 点击"添加跟踪记录"按钮
   - 选择跟踪方式：
     - 📞 电话
     - 💬 微信
     - 👥 面谈
   - 填写内容：
     ```
     示例：
     首次电话沟通，客户表示正在泥木阶段，
     预算8-10万，主要做窗帘和墙纸。
     下周三有时间到店。
     ```
   - 选择跟进结果：
     - ✅ 有意向（继续跟踪）
     - 📅 预约到店
     - ❌ 暂无意向

3. **设置下次跟进时间**
   - 填写"下次跟进时间"
   - 系统会在约定时间提醒您

**⏰ 时效要求**：
- ⚠️ **7天内必须有跟进记录**
- 🚫 超过7天无跟进：触发"无跟进预警"

---

#### 2.2 客户到店接待

**场景**：客户按约定时间到店

**操作步骤**：

1. **更新业务标签**
   - 在线索详情中添加标签："已到店"
   - 系统会自动记录到店时间

2. **详细沟通需求**
   - 了解具体空间（客厅/卧室/书房）
   - 确认产品类别（窗帘/墙纸/地板）
   - 记录特殊要求

3. **添加详细跟踪记录**
   ```
   示例：
   客户今日到店，带了户型图。
   需求：客厅窗帘（6米×2.8米），主卧墙纸（15平米）
   风格：现代简约
   预算：窗帘5000，墙纸4000
   ```

4. **标记客户级别**
   - A级：预算充足，决策快，近期成交
   - B级：有明确需求，预算合理
   - C级：仅咨询，预算模糊
   - D级：无明确需求

**操作提示**：
```
💡 A/B级客户 → 立即生成报价单
💡 C/D级客户 → 继续培育，定期跟进
```

---

### 阶段 3：报价与草签

#### 3.1 生成报价单

**前置条件**：客户需求明确

**操作步骤**：

1. **点击"生成报价单"**
   - 在线索详情页点击按钮
   - 或在报价单模块新建

2. **填写报价信息**
   - 项目名称：客户姓名 + 项目地址
   - 添加报价明细：
     ```
     示例：
     空间：客厅
     类别：窗帘
     产品：XX遮光布
     数量：18平米
     单价：280元/平米
     总价：5040元
     ```

3. **保存并发送**
   - 点击"保存报价单"
   - 通过微信/邮件发送给客户

4. **系统自动操作**
   - 线索业务标签自动添加："已报价"
   - 线索状态保持"跟踪中"

**⏰ 时效提示**：
- 30天内完成跟踪阶段（包括报价）
- 超过42天：扣5分

---

#### 3.2 客户确认草签

**场景**：客户口头同意或支付定金

**操作步骤**：

1. **点击"转为草签"**
   - 在线索详情页点击按钮
   - 确认对话框：请确认客户已明确同意

2. **填写草签信息**（可选）
   - 定金金额
   - 预计签约时间
   - 备注说明

3. **系统自动操作**
   - 状态变更为"草签（DRAFT_SIGNED）"
   - 背景色变为绿色
   - 进入下一阶段流程

**⚠️ 重要提醒**：
```
❌ 不要在客户仅表示"考虑"时就点击草签
❌ 草签后72小时内必须生成正式报价单
✅ 确保客户有明确付款意图再草签
```

**⏰ 时效要求**：
- 草签后**72小时内必须生成报价单**
- 超时：触发预警

---

### 阶段 4：测量与方案

#### 4.1 安排上门测量

**前置条件**：客户已草签

**操作步骤**：

1. **点击"安排测量"**
   - 线索状态变为"待测量"

2. **系统自动流程**
   ```
   待测量 → 测量中-待分配（调度分配测量师）
           ↓
        测量中-分配中（测量师接单）
           ↓
        测量中-待上门（测量师上门）
           ↓
        测量中-待确认（您确认测量结果）
   ```

3. **您的操作**
   - 等待测量师完成测量
   - 收到通知后查看测量结果
   - 点击"确认测量结果"或"驳回"

**⏰ 时效要求**：
- 点击安排测量后**48小时内调度必须分配**
- 测量完成后**48小时内您必须确认**
- 超时：触发预警

---

#### 4.2 方案确认

**场景**：设计师出图后需要您和客户确认

**操作步骤**：

1. **查看设计方案**
   - 在报价单详情中查看方案图
   - 或在线索"报价单"标签下查看

2. **与客户沟通**
   - 微信/到店展示方案
   - 确认客户是否满意

3. **确认方案**
   - 点击"确认方案"按钮
   - 状态变更为"待推单"

**操作场景分支**：
```
客户满意 → 点击"确认方案"
客户不满意 → 点击"驳回方案"→ 设计师修改

💡 驳回时务必注明客户具体意见
```

**⏰ 时效要求**：
- 方案出图后**72小时内与客户达成共识**
- 超时：触发管理层关注

---

### 阶段 5：推单与下单

#### 5.1 推单操作

**前置条件**：方案已确认

**当前状态**："待推单"（紫色）

**操作步骤**：

1. **点击"推单"按钮**
   - 填写推单备注（如有特殊要求）
   - 提交审核

2. **等待审核**
   - 状态变为"待下单"（黄色）
   - 交付部门会在48小时内审核

3. **审核结果**
   ```
   通过 → 状态变为"生产中"
   驳回 → 状态回到"待推单"，请修改后重新提交
   ```

**⏰ 时效要求**：
- 无强制时效（客户可能需要更多时间考虑）
- 但建议在7天内完成推单

---

### 阶段 6：订单跟踪

#### 6.1 生产与发货阶段

**您可见的状态**：
- 生产中（灰色）
- 备货完成（绿色）
- 待发货（红色）

**操作建议**：
```
💡 在"生产中"阶段：
   - 定期与客户联系
   - 告知进度
   - 维护客户关系

💡 在"备货完成"阶段：
   - 通知客户即将发货
   - 确认收货地址和联系人
   - 点击"整单发货"（如您有权限）
```

---

#### 6.2 安装阶段

**系统自动流程**：
```
安装中-待分配 → 安装中-分配中 → 安装中-待上门
    ↓                                    ↓
  调度分配                            安装师上门
                                         ↓
                              安装中-待确认（您审核）
```

**您的操作**：

1. **收到安装完成通知**
2. **查看安装照片**
   - 在线索详情的"附件"中查看
3. **审核安装结果**
   - 点击"审核安装结果"
   - 满意 → 通过
   - 不满意 → 驳回（必须填写原因）

**⚠️ 注意事项**：
```
❌ 不要未经实际验收就通过
✅ 必要时要求安装师返工
✅ 与客户确认满意度
```

---

### 阶段 7： 完成与售后

#### 7.1 订单完成

**状态**："已完成"（绿色）

**操作建议**：
```
💡 回访客户满意度
💡 请求好评/转介绍
💡 维护长期关系
```

**操作步骤**：

1. 点击顶部"预警"图标

3. 按优先级排序：

   - 🟡 测量逾期
4. 逐个点击处理

**处理模板**：
```
无跟进预警 → 立即添加跟踪记录
超时预警 → 说明原因 + 采取行动
```

| 测量中-待确认 | 48小时内确认或驳回 | 24小时 | 报警 |

### 8种预警类型及处理

#### 1. 🔴 跟踪逾期预警

**含义**：线索超过7天无跟进记录

**处理**：
```
→ 立即添加跟踪记录
→ 如客户失联，标记"暂停"并注明原因
```

---

#### 2. 🟠 已报价无草签预警

**含义**：已发报价单14天，客户未草签

**处理**：
```
→ 主动联系客户了解原因
→ 调整报价或产品方案
→ 如无希望，关闭线索
```

---

#### 3. 🟡 测量逾期预警

**含义**：安排测量后超过规定时间未完成

**处理**：
```
→ 联系调度查询进度
→ 催促测量师
→ 必要时向主管反馈
```

---

#### 4. 🔴 高意向客户无跟进

**含义**：A级/B级客户7天无跟进

**处理**：
```
→ 最高优先级处理
→ 立即联系客户
→ 更新状态或降级
```

---

#### 5. 🟠 预算超支预警

**含义**：报价金额超出客户预算30%

**处理**：
```
→ 重新核算报价
→ 调整产品配置
→ 与客户重新沟通
```

---

#### 6. 🟡 流失风险预警

**含义**：客户长时间停滞在某阶段

**处理**：
```
→ 分析停滞原因
→ 主动联系客户
→ 提供优惠或调整方案
```

---

#### 7. 🔴 竞品威胁预警

**含义**：客户提到竞争对手

**处理**：
```
→ 了解竞品方案和价格
→ 强调我方优势
→ 适度降价或增值服务
```

---

#### 8. 🟠 无7天跟进预警

**含义**：任意线索7天内无任何操作

**处理**：
```
→ 等同于"跟踪逾期"
→ 立即跟进或说明情况
```

## ❓ 常见问题与注意事项

### Q1：客户说"再考虑考虑"，我该怎么办？

**回答**：
```
✅ 不要立即草签
✅ 添加跟踪记录，注明"客户犹豫中"
✅ 设置3天后跟进提醒
✅ 询问具体顾虑（价格/质量/时间）
✅ 针对性提供解决方案
```

---

### Q2：测量师分配后一直不上门怎么办？

**回答**：
```
1. 查看状态是否为"测量中-待上门"
2. 在系统中查看测量师联系方式
3. 主动电话联系测量师
4. 如仍无响应，联系调度主管
5. 系统会在48小时后自动报警
```

---

### Q3：我误点了"草签"，怎么撤销？

**回答**：
```
❌ 系统不支持撤销
✅ 立即联系销售主管，说明情况
✅ 主管可以将状态改回"跟踪中"
⚠️ 频繁误操作会影响考核
```

---

### Q4：客户预算不足，无法成交，如何处理？

**回答**：
```
方案1：调整产品配置，降低报价
方案2：分期付款或优惠活动
方案3：暂停线索，备注"预算不足"
方案4：6个月后重新激活
❌ 不要强行成交导致客户投诉
```

---

### Q5：如何提高我的转化率？

**建议**：
```
✅ 快速响应：线索分配后24小时内联系
✅ 专业服务：详细记录客户需求
✅ 及时跟进：不要让线索"变冷"
✅ 精准报价：符合预算+展示价值
✅ 售后维护：成交后持续关系
```
