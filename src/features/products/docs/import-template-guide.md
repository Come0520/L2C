# 商品导入模板标准说明 (Product Import Template)

> 版本：v1.0 | 最后更新：2026-02-23
> 对应功能：商品库 → 批量导入商品

---

## 一键下载

模板文件（`.xlsx` 格式）可在商品管理页面点击「**批量导入** → 下载模板**」**获取，或直接访问路径：

```
/templates/product_import_template.xlsx
```

---

## 字段说明

| 列名 | 字段 | 类型 | 必填 | 说明 |
|:---|:---|:---|:---:|:---|
| 产品名称 | `name` | 文本 | ✅ | 商品完整名称，不能为空 |
| SKU型号 | `sku` | 文本 | ✅ | 唯一库存单位编号，全局不可重复 |
| 品类 | `category` | 枚举 | ✅ | 填写中文品类名（见下表），系统自动转换 |
| 零售价 | `retailPrice` | 数字 | ❌ | 面向客户的建议零售价，默认 0；单位：元 |
| 采购价 | `purchasePrice` | 数字 | ❌ | 供应商采购成本，默认 0；单位：元 |
| 描述 | `description` | 文本 | ❌ | 商品简介，最长 500 字 |

### 品类 (category) 枚举值

| 填写值（中文） | 说明 |
|:---|:---|
| 窗帘 | 挂帘、布艺帘等 |
| 卷帘 | 卷轴式遮光/半遮帘 |
| 百叶帘 | 横式/竖式百叶 |
| 纱帘 | 薄透纱幔 |
| 电动帘 | 含电机的智能帘类 |
| 罗马帘 | 折叠提拉式 |
| 配件 | 轨道、配套五金等 |
| 其他 | 以上分类外的商品 |

## 数据格式规范

### SKU 编号
- 全字母或字母+数字组合，推荐格式：`CL-2024-001`
- **同一租户内唯一**，导入时系统自动检重，冲突会中止并报告错误行

### 价格字段
- **纯数字**，不含货币符号（`¥`）和千分位逗号
- 支持小数点，精确到分（如 `298.50`）
- 留空或填 0 均视为「价格待定」

---

## 导入流程

```
1. 下载模板 → 2. 填写数据 → 3. 上传文件 → 4. 系统预检 → 5. 确认导入
```

1. **预检阶段**：系统逐行校验数据，输出错误报告（含行号和原因）
2. **事务导入**：预检通过后，批量写入使用数据库事务保证原子性
3. **部分失败**：若任一行数据不合规，整批导入将回滚，请修正后重新上传

---

## 常见错误与解决方案

| 错误信息 | 原因 | 解决办法 |
|:---|:---|:---|
| `名称不能为空` | 产品名称列为空 | 补填产品名称 |
| `SKU不能为空` | SKU列为空 | 补填 SKU 编号 |
| `品类格式不正确` | 填写了非法品类值 | 参照上方枚举表填写 |
| `SKU重复` | 与库中已有商品冲突 | 修改 SKU 或跳过已存在商品 |
| `零售价格式错误` | 含有货币符号或逗号 | 改为纯数字，如 `288` |
